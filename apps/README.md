# Apps

| App | Type | Purpose |
| --- | --- | --- |
| `example-svelte-css/` | Example | Consumer-style Svelte 5 app using `@astryxdesign/svelte/styles.css` and plain CSS only |
| `example-svelte-tailwind/` | Example | Consumer-style Svelte 5 + Tailwind v4 app using `@astryxdesign/svelte` and token utilities |
| `svelte-storybook/` | Visual review | Storybook-like Svelte visual surface for components, lab surfaces, and Vega wrappers |
| `svelte-docs/` | Docs | Lightweight static docs shell for the Svelte release line |

Run from the repository root:

```bash
pnpm -F @astryxdesign/example-svelte-css build
pnpm -F @astryxdesign/example-svelte-tailwind build
pnpm -F @astryxdesign/svelte-storybook build
pnpm -F @astryxdesign/svelte-docs build
```
